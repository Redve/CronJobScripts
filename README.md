# CronJobScripts
This is my repo for running smaller tasks such as checking prices for certain products I want when they are not available on Prisjakt for example.
## Dell XPS 15
This checks for the price of the Dell XPS 15 base conf and send an email of the price has changed. The price is check on Dells.
